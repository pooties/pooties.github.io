  67. -Changing Your Icons-
  68. The easiest way to change your program icons individually, without any tools or software, is to edit a shortcut. What I recommend doing is navigating to the programs root directory, and creating the shortcut there so there is no confusion. Creating a shortcut also gives you the added bonus of naming it whatever you want, and even setting launch options. Right click on the Icon>Properties>Change Icon. Use your desired .ico file. Pin this shortcut to your taskbar/start menu! For other icons (library icons, task tray icons) please see the windows software section for tools to help you change them. Its a lot faster and easier than going through registries and replacing files manually. Look for 7tsp or 7conifier for icon packs, and browse for icon packs in the Icon section!

  85. -Making sure your taskbar is aligned properly, and slim as possible-
  86. Ever install appows and notice the icons or the start orb seem to be a little misaligned? This is just a glitch with the taskbar that is easily solved by extending it. Unlock your task bar, and drag it to the top of your screen. You can do this by left clicking and holding the bar near the center, and moving it to the edges of the screen. Drag it to the top along the edges this way. Once its on top, hover over the middle of the bar near the bottom border until you see the resize option for height ?. Drag it down, extending it to double the normal height. Move it back down to the bottom of your screen, then size it back down using the same ? option. Lock your task bar. You should be at the correct size and alignment now.

  251. ***Font Rendering***
  252. Fonts appearing to have a greenish tint or look blurry in Firefox/Bblean/Windows? This is due to an acceleration error, probably caused by subpixel rendering or fonts not being supported in smaller sizes. Windows Cleartype also causes issues with EVERY browser, and overall is complete shit compared to Linux or Mac/OSX font rendering. Read about Cleartype for details on how it works. Consider switching fonts if none of the fixes below work for you, or try and get a version of your font that is NOT ".tff" or true type.
  253.  
  254. The best and easiest option in my opinion is to install Mactype or gdipp. These are services that replace Windows Cleartype font rendering with Mac/OSX or Linux font rendering. Mactype is more customizable and has bolder bolds, while Gdipp is easier to use and works immediately upon install. If you are using a bitmap font, font rendering is not necessary. Cleartype should be off and you should not need to use Mactype or gdipp. This is only true if you are using only bitmap fonts. If you are using a mix of fonts, I recommend Mactype since its easier to edit your exclusions.
  255. Go here for details.

  257. Mactype recently got its installer taken offline and the project on github has no links to an installer.
  258. spoonm offered to host it on his website: here.

  260. Recommended fixes for Firefox/Nightly:
  261. In Firefox, type "about:config" in the address bar and hit Enter. Continue after the warning.
  262. In the filter box at the top, type "gfx.direct2d.disabled". This value will be set to false by default.
  263. Right click that line and select Toggle. This will change the value to true. Restart Firefox. You may also want to toggle "Use hardware acceleration when available" in your Firefox settings. This is under Options>Advanced>General tab.
  264. Try the Anti-Aliasing Tuner too, if you feel the need for more tweaking.

  414. ***Auto Hot Key Scripts***
  415. Website - Tutorial
  416. Auto Hot Key allows you to program hotkeys, and keystrings, and commands in an easy to understand format. You can then run these scripts on windows by compiling it (creating a .exe) or running it directly through Auto Hot Key (running as .ahk). It's been most commonly used to "tile" or manage/resize windows, but has several other uses. You can remove borders and captions from windows, set up keystrings (i.e.: ty becomes "thank you" automatically), and even launch programs. It saves time, and its great for those who use long passwords or want a more efficient way to manage the way Windows functions.

  436. W= Distance of the window from the left side of the screen
  437. X= Distance of the window from the top of the screen
  438. Y= Total width of the window
  439. Z= Total height of the window
  440.  
  441. Like so:
  442. NumpadHome:: (defined hotkey)
  443.             WinMove,A,, 20,29,1024,512 (defined window size and position)
  444.
